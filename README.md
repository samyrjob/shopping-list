A gamified shopping list progressive web app built with Blazor WebAssembly.

## Features

**Core**
- Add and remove items from your shopping list
- Search and filter items
- Mark items as purchased with one tap
- Clear all purchased items at once

**Gamification**
- Points system rewarding completed items
- Daily streak tracking
- Level progression

**Technical**
- Offline-first architecture with service workers
- Client-side storage using LocalStorage
- Installable on mobile devices as a standalone app

## Stack

- Blazor WebAssembly (.NET 8)
- Azure Static Web Apps
- LocalStorage API

## Install on Mobile

Open the app in Chrome, tap the browser menu, select "Add to Home Screen".
